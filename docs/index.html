Kalendario - Browser Extension

Kalendario is an extension for browsers (Chrome, Opera, Vivaldi, Brave ...) that enhances the functionality of Google Calendar by displaying descriptions under events in the agenda and search pages.

This simplifies the work with the application - you don't have to click the specific event to show its description. Great for keeping a workout log, a daily journal or weekly/monthly/quarterly reviews.
